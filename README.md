# SMHI-API

## Hämta data från SMHI API

Ett enklet sätt att hämta data från SMHI API.

## Default paramterna:

- Datum och tid
- Temperatur
- Vindhastighet
- Lufttryck
- Riska för åska/blixt
- Nedebörd
- Vädersymbol


## Ändra på paramterna:
Du kan ändra på vilka paramtera som skrivs ut i filen createElement.js.
Där kan du även ändra på HTML-formateringen. 

Lista över de olika parameterna hittar du i länken 
[SMHI](https://opendata.smhi.se/apidocs/metfcst/parameters.html)
